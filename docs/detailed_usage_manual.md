## 2. 潜水员端使用流程

### 2.1 日历工作台

1. 登录潜水员账号。
2. 进入日历工作台。
3. 点击日期查看当天作业情况。
4. 查看作业批次、上传案例、任务到期和审核状态。
5. 点击最近上传案例查看图片、AI 增强、识别结果和证据链。

### 2.2 水下模式

1. 进入“水下模式”。
2. 选择潜点和关联任务。
3. 填写天气、能见度、水流、深度等环境信息。
4. 使用地图选点或潜点默认坐标记录作业位置。
5. 点击开始作业完成签到。
6. 上传水下照片或手机拍照。
7. 填写清理状态、现场标签和备注。
8. 提交后系统生成案例和垃圾身份证。
9. 结束作业后页面刷新，可重新签到或进入岸上模式。

### 2.3 岸上模式

1. 进入“岸上模式”。
2. 在日历中选择需要复核的日期。
3. 查看当天上传的案例。
4. 查看原图、AI 增强图和 AI 识别图。
5. 根据实物垃圾修正分类、数量、重量和清理状态。
6. 填写称重方式和修正说明。
7. 点击确认入库。
8. 如已确认，可重新提交复核。
9. 填写志愿者心声，系统会提取语义标签和风险关键词。

### 2.4 我的任务

1. 查看管理员派发的清理、巡查或复查任务。
2. 根据任务中的潜点、坐标、优先级和说明执行作业。
3. 完成后点击提交完成。
4. 等待管理员确认。

## 3. 管理员端使用流程

### 3.1 治理态势总览

管理员登录后可查看：

- 治理地图。
- 案例数、垃圾件数、清理重量和待审核数。
- 活动趋势图。
- 潜水员贡献表。
- 材质分布。
- 近期任务。
- 志愿者心声洞察。

### 3.2 潜水员活动管理

1. 点击“潜水员活动管理”。
2. 选择某个潜水员。
3. 查看该潜水员的工作日历。
4. 点击具体日期查看当天作业批次和案例。

### 3.3 垃圾管理

1. 进入“垃圾管理”。
2. 选择全部、未清理或已清理。
3. 点击垃圾身份证查看详情。
4. 查看原图、AI 增强图、AI 识别图和证据链。
5. 修正分类、数量、重量、风险和清理状态。
6. 点击审核通过。
7. 审核后按钮状态变为已确认审核，并保留重新审核入口。
8. 对未清理垃圾可生成清理任务。

### 3.4 任务分配

1. 进入“任务分配”。
2. 查看进行中任务地图。
3. 可选择已有未清理垃圾作为目标。
4. 也可以在地图上点击新的任务地点。
5. 填写潜点、潜水员、任务类型、优先级、风险等级、截止日期和说明。
6. 点击创建任务。
7. 任务会进入潜水员端“我的任务”。

### 3.5 漂流溯源沙盘

1. 进入“漂流沙盘”。
2. 选择分析对象：单个垃圾或区域范围。
3. 选择分析模式：反向溯源、未来扩散或两者同时分析。
4. 单件模式选择垃圾身份证。
5. 区域模式在地图上选择区域中心并设置半径。
6. 地图上会显示垃圾点，便于选择密集区域。
7. 点击运行沙盘分析。
8. 查看溯源轨迹、扩散轨迹、概率区、区域样本、数据来源和行动建议。

### 3.6 报告中心

1. 进入“报告中心”。
2. 选择报告类型和范围。
3. 点击生成多智能体报告。
4. 右侧查看报告草稿。
5. 检查多 Agent 结论、KPI、任务联动、漂流引用和志愿者反馈。
6. 点击确认发布。
7. 可归档或打印/导出课件。

### 3.7 人员管理

1. 进入“人员管理”。
2. 输入用户名、姓名、角色、组织、初始密码和手机号。
3. 点击创建账号。
4. 新账号可使用设置的密码登录。

## 4. 手机端使用建议

- 手机和电脑需要处于同一局域网。
- 使用浏览器访问电脑局域网 IP。
- 潜水员优先使用水下模式和岸上模式。
- 手机端底部为横向导航，可左右滑动。
- 地图支持拖动、缩放和点击选点。

## 5. 演示流程建议

推荐演示顺序：

1. 管理员登录，展示治理态势总览。
2. 潜水员登录，展示日历和水下模式。
3. 上传一张案例图片，生成垃圾身份证。
4. 在岸上模式完成分类和称重复核。
5. 管理员审核垃圾身份证。
6. 管理员基于垃圾或新地图点创建任务。
7. 运行漂流沙盘。
8. 生成多智能体报告并确认发布。
