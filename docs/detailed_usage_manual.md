# SeeSea 详细使用说明

## 1. 文档目的

本手册面向实际操作人员，覆盖：

- 管理员端操作
- 潜水员端操作
- 手机接入方式
- 演示流程建议
- 常见问题处理

## 2. 启动与访问

### 2.1 启动命令

```bash
cd <project-root>
conda activate clean_underwater_demo
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

其中 `<project-root>` 为项目根目录。

### 2.2 访问入口

- 登录页：`http://127.0.0.1:8000/`
- 管理员端：登录后自动进入
- 潜水员端：登录后自动进入

### 2.3 手机访问

手机与电脑在同一局域网时，通过：

- `http://你的电脑局域网IP:8000/`

不要使用 `https://`。

## 3. 账号说明

- 管理员：`admin / admin123`
- 潜水员 A：`diver_a / diver123`
- 潜水员 B：`diver_b / diver123`

## 4. 潜水员端操作说明

### 4.1 登录后首页

潜水员端首页包含：

- 个人信息
- 统计卡片
- 新建案例表单
- 我的上传实例列表
- 当前选中案例详情

### 4.2 新建案例字段说明

需填写或确认：

- 实例标题
- 潜点名称
- 拍摄日期
- 位置描述
- 纬度
- 经度
- 天气
- 能见度
- 人工表单备注
- 志愿者语义反馈

### 4.3 位置填写方式

支持两种方式：

#### 方式 A：读取手机定位

点击 `读取手机定位` 后：

- 若浏览器允许定位，则自动写入经纬度
- 若浏览器拒绝，则会提示改用地图选点

#### 方式 B：地图选点

点击 `地图选点` 后：

1. 展开地图
2. 在地图上点击位置
3. 系统自动写入经纬度
4. 如已有坐标，可点击 `定位到当前坐标`

### 4.4 上传图片方式

支持两种方式：

#### 方式 A：拍照上传

- 适合手机端现场采集
- 系统优先调用手机摄像头或后置相机

#### 方式 B：选择图片

- 适合已拍好的图片
- 可从相册或电脑文件夹中选择

图片选中后会显示预览。

### 4.5 人工表单条目

点击 `新增表单条目` 可添加一条垃圾记录，字段包括：

- 垃圾名称
- 数量
- 状态（待清理 / 已清理）

### 4.6 提交后系统自动完成的处理

提交后系统会依次完成：

1. 图片保存
2. 图像增强
3. AI 垃圾识别
4. 人工表单核对
5. 志愿者反馈语义分析
6. 扩散路径模拟
7. 案例报告生成

### 4.7 潜水员端可查看结果

每个案例详情中可查看：

- 原图
- 增强图
- AI 识别图
- 识别与核对结果
- 案例分析报告
- 管理员反馈
- AI 补充洞察

### 4.8 删除案例

步骤：

1. 选择案例
2. 点击 `删除本案例`
3. 确认删除

删除时会同步清理该案例独占的图片资源。

## 5. 管理员端操作说明

### 5.1 首页内容

管理员首页包含：

- 指标卡片
- 垃圾类别分布
- 材质分布
- 来源类型分布
- 月度上传趋势
- 潜点地图
- AI 指挥简报
- 同源预警与运营提醒
- 案例列表
- 潜水员列表
- 潜水员新增表单

### 5.2 筛选功能

管理员可按以下条件筛选案例：

- 关键词
- 风险等级
- 审核状态
- 清理状态
- 潜水员

筛选会同步影响：

- 地图
- 统计图
- 列表
- AI 指挥简报
- 风险预警

### 5.3 模型配置状态

管理员端会显示：

- 当前提供方
- 模式
- 视觉模型
- 文本模型
- 超时配置
- 重试次数
- 输出上限
- 当前接口地址
- 最近成功时间
- 最近错误

### 5.4 LLM 连通性测试

点击 `测试 LLM 连通性` 后，系统会直接返回：

- 是否连接成功
- 当前模型
- 接口地址
- 失败原因

这个功能用于快速判断：

- Key 是否有效
- 接口是否可达
- 当前文本模型是否能正常调用

### 5.5 案例详情页

管理员可查看：

- 原图 / 增强图 / AI 识别图
- AI 与人工核对对比
- 污染源分析
- 扩散路径点位
- 相似案例
- 错误信息

### 5.6 工作流管理

管理员可修改：

- 审核状态
- 清理状态
- 优先级
- 管理员备注

### 5.7 重新生成分析

当案例需要重新跑 AI 分析时，可点击：

- `重新生成分析`

### 5.8 数据导出

支持：

- 当前筛选范围 CSV 导出
- 单案例 Markdown 导出
- 单案例 JSON 导出

### 5.9 潜水员管理

管理员可：

- 查看潜水员汇总
- 查看高频潜点和主要来源
- 新增潜水员账号

## 6. 演示建议

### 6.1 5 分钟版本

1. 管理员登录
2. 展示地图、趋势、预警和 AI 指挥简报
3. 切换到潜水员端
4. 拍照上传一个案例
5. 回到管理员端查看新增案例
6. 展示识别图、核对结果和报告

### 6.2 10 分钟版本

1. 登录管理员端，介绍全局架构
2. 展示模型配置和 LLM 测试
3. 展示地图、来源分布和简报
4. 切换潜水员端做现场采集
5. 演示地图选点
6. 提交案例
7. 回到管理员端查看案例详情和审核流
8. 结合技术报告讲创新性、可实施性和可持续性

## 7. 常见问题

### 7.1 手机打不开页面

检查：

1. 手机和电脑是否在同一 Wi-Fi
2. 启动命令是否使用 `--host 0.0.0.0`
3. 是否使用 `http://` 而不是 `https://`
4. 防火墙是否拦截端口 8000

### 7.2 手机定位失败

原因通常是浏览器安全策略。

处理：

- 直接使用 `地图选点`

### 7.3 LLM 未成功调用

先点击 `测试 LLM 连通性`，并查看模型配置区域中的最近错误信息。

### 7.4 数据库正忙

说明当前有较重任务正在处理，请稍后再试。
