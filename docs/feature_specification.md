# SeeSea 详细功能说明

## 1. 产品定位

SeeSea 是一套面向海洋垃圾治理行动的双端智能协作系统。它的核心目标不是单纯识别垃圾，而是建立一条从前线采集、AI 处理、人工核对到后台管理和项目复盘的完整数据链路。

系统包含两类用户：

- 潜水员：负责现场采集与提交
- 管理员：负责审核、汇总、导出与指挥支持

## 2. 功能模块总览

### 2.1 登录与权限隔离

- 管理员 / 潜水员双角色登录
- 角色独立 token 鉴权
- 同一浏览器中双角色登录互不覆盖

### 2.2 潜水员端采集功能

- 手机拍照上传
- 相册或本地文件选择
- 图片预览
- 潜点、日期、位置、天气、能见度填写
- 人工表单条目录入
- 志愿者反馈录入
- 删除本人案例

### 2.3 定位与地图选点

- 浏览器定位优先
- HTTP 场景下支持地图选点降级
- 点击地图自动回写经纬度

### 2.4 图像增强

- 对原始水下图片进行增强
- 提升可辨识度
- 输出增强图用于展示和识别

### 2.5 垃圾识别

- 传统视觉规则识别
- 可选本地 ONNX 检测器融合
- 可选远程视觉模型辅助识别
- 输出垃圾类别、材质、来源活动、数量、尺寸、状态、置信度

### 2.6 OCR 与包装线索识别

- 从图像中提取包装文字
- 将 OCR 文本与包装匹配结果作为识别证据
- 增强垃圾来源判断能力

### 2.7 人工表单核对

- 自动对比 AI 识别结果和人工表单
- 输出 AI 总数、人工总数、是否一致
- 标出漏填项和数量偏差项

### 2.8 志愿者反馈语义分析

- 分析垃圾密集度
- 分析环境复杂度
- 分析潜在安全风险
- 输出风险等级

### 2.9 扩散路径模拟

- 根据潜点与来源类型生成规则驱动的扩散轨迹
- 输出轨迹点列表
- 在管理员端表格化展示

### 2.10 单案例分析报告

- 自动生成摘要
- 自动生成污染源分析
- 自动生成扩散分析
- 自动生成核对说明
- 自动生成行动建议与传播文案

### 2.11 管理员看板与工作流

- 指标卡片
- 类别/材质/来源分布
- 月度趋势
- 地图总览
- AI 指挥简报
- 风险预警
- 审核状态、清理状态、优先级、备注维护

### 2.12 潜水员汇总

- 潜水员历史案例汇总
- 高频潜点分析
- 主要来源分析
- 汇总报告生成

### 2.13 导出能力

- 单案例 Markdown 导出
- 单案例 JSON 导出
- 当前筛选范围 CSV 导出

### 2.14 LLM 连通性测试

- 管理员可一键测试当前 SiliconFlow 文本模型可用性
- 返回成功/失败、模型名、接口地址和错误原因
- 便于现场排障

### 2.15 面向评审材料的分析能力

系统还提供一套专门面向技术报告和答辩材料的分析方法，用于总结：

- 公益价值
- 可实施性
- 成本与运营负担
- 可持续性
- 技术扩展潜力

这部分内容主要体现在技术报告和 PPT 中，不在管理员首页直接展示。

## 3. 系统特征

- 图像即数据
- 双端协同
- AI 与人工双向核验
- 本地规则 + 本地模型 + 云端模型分层运行
- 失败可降级，适合比赛现场演示
- 低部署成本，适合公益机构试点

## 4. 面向评审的价值总结

### 技术实现难度

- 多阶段 AI 处理链路
- 多源结果融合
- 双角色工作流
- 可视化与导出并存

### 创新性

- 垃圾“身份证”概念
- 图像到结构化数据的转化
- 案例级到行动级的决策支持

### 功能完整度

- 覆盖采集、识别、核对、分析、审核、导出全流程

### 可实施性

- FastAPI + SQLite + 浏览器端地图的轻量架构
- 不依赖专用硬件
- 具备本地降级能力

### 可持续性

- 数据可长期积累
- 模型接口可替换
- 可逐步升级到更强的识别、OCR 和轨迹模型
