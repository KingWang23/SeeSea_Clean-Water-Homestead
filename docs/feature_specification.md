# SeeSea 详细功能说明

## 1. 产品定位

SeeSea 是一套服务海洋垃圾治理行动的双端协作系统，目标是把潜水员拍摄到的水下图像、人工表单和志愿者反馈转化为可核对、可追踪、可展示、可持续积累的数据资产。

系统由两部分组成：

- 潜水员端：负责移动端采集
- 管理员端：负责审核、分析、统筹和导出

## 2. 核心功能模块

### 2.1 登录与角色隔离

- 支持管理员、潜水员双角色登录
- 采用角色独立 token 鉴权
- 同一浏览器中管理员与潜水员不会互相覆盖登录态

### 2.2 潜水员端移动采集

- 支持手机拍照上传
- 支持从相册选择图片
- 支持图片预览
- 支持填写潜点、日期、天气、能见度、位置描述
- 支持人工表单录入
- 支持志愿者语义反馈录入
- 支持删除本人提交的案例

### 2.3 定位与地图选点

- 优先调用浏览器定位
- 若移动端在 HTTP 环境下无法直接定位，则自动降级为地图选点
- 点击地图即可自动写入经纬度
- 适合答辩演示和真实户外场景

### 2.4 水下图像增强

- 对上传原图执行图像增强
- 输出增强图供识别和展示使用
- 解决偏色、模糊、亮度不足、局部细节不清等问题

### 2.5 AI 垃圾识别

- 传统视觉规则识别
- 可选本地 ONNX 检测器融合
- 可接入 SiliconFlow 视觉模型
- 输出类别、材质、来源活动、尺寸级别、数量、置信度
- 同时生成证据链字段，记录识别依据来源

### 2.6 OCR + 包装线索识别

- 对图像中的包装文字进行 OCR
- 结合包装和外观特征辅助判断垃圾来源
- 为“垃圾身份证”提供文本证据

### 2.7 人工表单核对

- 自动对比 AI 识别结果和人工录入结果
- 输出 AI 总数、人工总数、是否一致
- 标出漏填、错填和数量偏差

### 2.8 志愿者反馈语义分析

- 对志愿者文本进行语义拆解
- 提取垃圾密集度、环境复杂度、安全风险
- 自动给出风险等级

### 2.9 扩散路径模拟

- 根据潜点位置、来源类型和规则逻辑生成扩散路径点位
- 输出轨迹点列表
- 在管理员端可视化展示

### 2.10 案例级综合报告

- 自动生成单案例分析报告
- 内容包含摘要、污染源分析、扩散分析、核对说明、行动建议、传播文案、创新建议
- 支持管理员重新生成
- 支持 Markdown / JSON 导出

### 2.11 管理员工作流

- 审核状态管理
- 清理状态管理
- 优先级管理
- 管理员备注
- 案例删除

### 2.12 管理员总览看板

- 核心指标卡片
- 垃圾类别分布
- 材质分布
- 来源分布
- 月度上传趋势
- 地图分布
- 实例列表
- 潜水员列表

### 2.13 AI 指挥简报

- 针对当前筛选范围自动生成行动摘要
- 输出重点潜点、传播建议、数据质量关注点、下一步动作
- 支持远程 LLM 和本地规则双模式

### 2.14 公益成效与实施评估

系统提供一套面向评审材料和技术文档的分析方法，自动把当前项目数据转成：

- 估算清理重量
- 自动化覆盖率
- 数据可信度
- 节省志愿者工时
- 公益价值摘要
- 成本与运营负担评估
- 可持续运行建议

这部分内容主要服务于技术导师和公益导师答辩材料、技术报告和 PPT 分析，不在管理员首页直接展示。

### 2.15 LLM 连通性测试

- 管理员可一键测试当前 SiliconFlow 配置
- 直接返回成功或失败
- 显示模型、接口、错误原因
- 便于现场排障

## 3. 导出与数据沉淀

- 单案例 Markdown 导出
- 单案例 JSON 导出
- 全局筛选后 CSV 导出
- 可直接用于项目周报、月报、研究记录和公益传播素材

## 4. 系统特色

- 图像即数据
- 双端协同
- 本地规则 + 本地模型 + 云端模型三层架构
- 失败可降级，演示稳定
- 可快速部署在公益机构现有电脑上

## 5. 面向评审的价值总结

- 技术实现难度：多模型融合、双端协作、证据链与扩散模拟
- 创新性：垃圾身份证、图像到数据、公益流程数字化
- 功能完整度：采集、增强、识别、核对、审核、导出、简报全流程闭环
- 可实施性：轻量部署、低硬件门槛、支持 API 不可用时本地降级
- 可持续性：长期沉淀潜点和来源数据，便于形成年度治理资料库
