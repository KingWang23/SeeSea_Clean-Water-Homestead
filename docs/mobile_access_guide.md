# SeeSea 手机访问说明

SeeSea 潜水员端面向手机使用场景设计，手机浏览器可以直接访问本地部署的网页原型。

## 1. 网络要求

手机和运行 SeeSea 的电脑需要处于同一个局域网。

可用网络：

- 同一个 Wi-Fi。
- 电脑连接手机热点。
- 同一个有线/无线局域网。

不建议：

- 手机使用移动流量直接访问电脑本地服务。
- 使用公司或校园访客网络，可能会禁止设备互访。

## 2. 启动服务后查看访问地址

本地原型启动后会监听：

```text
0.0.0.0:8000
```

电脑端可以访问：

```text
http://localhost:8000
```

手机端需要访问电脑的局域网 IP，例如：

```text
http://192.168.1.23:8000
```

移动端入口：

```text
http://192.168.1.23:8000/mobile
```

## 3. 查看电脑局域网 IP

Linux / Ubuntu 可使用：

```bash
hostname -I
```

常见局域网 IP 形态：

- `192.168.x.x`
- `10.x.x.x`
- `172.16.x.x`

## 5. 手机端界面特点

- 底部横向导航，适合单手操作。
- 水下模式支持拍照/选择照片。
- 地图支持拖动、缩放和点击选点。
- 岸上模式支持按日期复核案例。
- 表单和卡片已针对窄屏压缩布局。

## 6. 常见问题

### 手机打不开页面

检查：

1. 手机和电脑是否在同一网络。
2. 地址是否使用 `http://`，不是 `https://`。
3. IP 是否为电脑局域网 IP。
4. 服务是否已经启动。
5. 防火墙是否放行端口。

Ubuntu 放行端口示例：

```bash
sudo ufw allow 8000/tcp
```

### 手机能打开但地图加载慢

地图瓦片需要联网访问高德地图服务。检查电脑或手机网络是否可访问外网。

### 手机无法自动定位

浏览器定位通常要求 HTTPS 或明确授权。本系统提供地图选点作为降级方式，可以手动选择潜点或垃圾点坐标。
