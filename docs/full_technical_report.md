# SeeSea 完整技术报告摘要

## 1. 背景

“洁净水下家园”项目已经积累大量潜水员清理行动数据，但传统流程中，水下垃圾识别、溯源分析、数据管理和成果展示仍依赖人工。SeeSea 面向这一痛点，构建从现场采集到治理决策的多智能体 AI 系统。

## 2. 系统目标

SeeSea 的目标是实现：

```text
水下采集 -> AI 识别 -> 人工复核 -> 垃圾身份证 -> 漂流溯源
       -> 任务派发 -> 清理确认 -> 公益影响评估 -> 报告发布
```

## 3. 需求覆盖

系统覆盖项目方核心技术需求：

- 垃圾溯源 AI 分析。
- 多维协作线上平台。
- 水下图像增强。
- 垃圾来源辨识。
- 志愿者反馈语义分析。
- 沉浸式数据可视化。

同时覆盖潜水作业流程要求：

- 入水前签到和安全确认。
- 水下拍照和定位。
- 上岸称重。
- 垃圾分类计数。
- 活动数据汇总和报告生成。

## 4. 核心技术路线

### 4.1 响应式双端系统

潜水员端和管理员端共用响应式 Web 技术，适配手机、电脑和主机端。

### 4.2 PostgreSQL + PostGIS

PostGIS 用于管理潜点、垃圾、任务和区域分析的空间数据，为后续海洋环境数据接入和 GIS 统计提供基础。

### 4.3 AI 证据链

系统把每个 AI 判断拆解为可追溯证据：

- 图像增强证据。
- 目标检测证据。
- OCR 证据。
- VLM 分类证据。
- 风险融合证据。
- 人工复核证据。

### 4.4 OpenDrift 漂流沙盘

漂流分析采用 OpenDrift / 拉格朗日粒子追踪思路，可从单件垃圾和区域样本出发分析来源与扩散。

### 4.5 多智能体报告

报告中心由多个 Agent 协同生成：

- 视觉分析 Agent。
- 证据链审计 Agent。
- 溯源推理 Agent。
- 安全评估 Agent。
- 行动规划 Agent。
- 公益影响 Agent。
- 志愿者反馈 Agent。

## 5. 创新点

### 5.1 垃圾身份证

每件垃圾都有唯一编号和完整档案，支持长期追踪和治理复盘。

### 5.2 可信 AI 证据链

AI 判断不再是黑盒输出，而是可以解释、复核和沉淀为训练样本。

### 5.3 漂流溯源沙盘

把地图升级为项目特色，从空间分布展示进一步支持污染源判断和治理优先级决策。

### 5.4 公益行动闭环

未清理垃圾可以转为任务，志愿者反馈可以转为复查，清理成果可以转为报告和传播材料。

## 6. 技术效果

- 提高水下垃圾数据采集效率。
- 提高垃圾分类和来源判断可信度。
- 建立可复核的公益治理数据资产。
- 支撑区域治理、长期趋势分析和公众传播。
- 可逐步升级为多机构、多潜点、多海域的海洋垃圾治理数字底座。
